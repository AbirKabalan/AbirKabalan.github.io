PROJECT

Book Rating System

A web application that is accessible through a web interface, providing unlimited Arabic and international books to rate and to discuss about by users.
It was built and designed using HTML5, PHP, MySQL as well as some JavaScript.

Application features:

  • * An authenticated user can add, edit and delete a book.

  • * Users can rate a book, as well as leave their comments and reply to other users.

  • * Without his activation by an admin, a user, although registered, is considered as a simple visitor.

  • * An admin can activate, ban, delete and add a user, delete a book or a post in discussion.

  • * Users can send reports that the admin generate.
